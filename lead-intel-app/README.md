# LeadIntel

A premium sales intelligence suite.

## Running the App
1. Open a terminal in this folder.
2. Run `.\start.bat`.
3. Open **[http://localhost:5173](http://localhost:5173)** in your browser.

## Features
- **Instant Logo Fetching**: Uses Clearbit to grab brand assets.
- **Email Discovery**: Integrated logic for Tomba/Hunter (Demo mode enabled).
- **Phone Intelligence**: Validates lead numbers via numverify patterns.
- **Glassmorphism UI**: High-fidelity dark mode design.

## API Setup
Edit `server/.env` to add your real keys for production data.
