## üöÄ Features

### üîç Intelligent Discovery Pipeline
- **Smart Intent Recognition**: Uses natural language processing heuristics to distinguish between homeowners asking for help ("I need sod") and professionals selling services ("We install sod").

## üì¶ Installation

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/PINNACLEAISOLUTIONS/MIAMILOVESGREENSCRAPER.git
    cd MIAMILOVESGREENSCRAPER
    ```

2.  **Install Dependencies**
    ```bash
    pip install -r requirements.txt
    cd lead-intel-app && npm install
    ```

3.  **Run the Scraper Dashboard**
    ```bash
    cd LandscapeLeadsFL
    python start.bat
    ```

4.  **Access the Dashboard**
    Open `http://localhost:5005` to view live leads.
